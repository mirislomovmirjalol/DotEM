# `export` Command

Export command is used to export all data from DotEM storage to a single JSON file.

Usage:

```bash copy
.em export [flags] <file_path>
```
Examples:
```bash copy
.em export /path/to/file.json
```
Options:

-h, --help: Help for export command.

Output Format:
```json
{
  "project1": "value1",
  "project2": "value2"
}
```
